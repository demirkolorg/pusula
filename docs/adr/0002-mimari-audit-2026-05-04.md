# ADR-0002 — Mimari Sözleşme Senkronizasyonu

## Bağlam
Plan dokümanı Next.js 15, gerçek proje 16.2.4 kullanıyor. README boilerplate kalmış. `.nvmrc`, `engines`, `packageManager`, CI workflow, `CHANGELOG.md` eksik.

## Karar
- `docs/plan.md` Next.js 16 olarak güncellendi.
- `README.md` Pusula'ya özgü yeniden yazıldı (Bun, port 2500, kurulum adımları).
- `.nvmrc` eklendi (Node 22 LTS).
- `package.json#engines` ve `packageManager` eklendi.
- `CHANGELOG.md` Keep a Changelog formatı, Türkçe.
- `.github/workflows/ci.yml`: install → typecheck → lint → test → build (Kontrol Kural 104).

## Sonuç
Yeni geliştiriciler doğru stack ile başlar. CI her PR'da güvence sağlar.
